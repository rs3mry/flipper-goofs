## How to Use

1. Copy the desired payload folders into the `badusb` directory on your Flipper Zero SD card.
2. Disconnect the SD card and insert it into your Flipper Zero.
3. Open the **BadUSB/BadKB** application.
4. Select the appropriate script for your target operating system.
5. Connect the Flipper Zero to the target device and press **Run**.
