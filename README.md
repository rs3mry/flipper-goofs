# flipper-goofs

A collection of harmless BadUSB scripts and keyboard macros designed purely for practical jokes. These are simply automated keystrokes intended to play harmless visual pranks on unattended screens. 

## DISCLAIMER

This repository is strictly for comedic purposes and UI automation testing. These scripts do not alter system files, steal data, bypass security, or perform any malicious actions. They simply emulate a keyboard pressing keys very fast to open funny pictures or websites. I am not responsible if your friends get annoyed at you. Please prank responsibly.

## HOW TO USE

1. Download the .txt files from this repository.
2. Place them in your Flipper Zero's MicroSD card under the badusb/ directory.
3. Plug the device into an unattended computer.
4. Run the script and walk away.
